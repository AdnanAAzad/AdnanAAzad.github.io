C • Temperature Control System

Windows Forms Application; A program that connects with a DE1 board to interface with a temperature sensor that sends and receives binary/HEX data values to indicate temperature changes which is displayed in the windows form application as well as the 7-Segment displays on the board. Alternatively, temperature can be simulated using the on-board buttons which is then updated accordingly in the program as well as 7-Segment displays. All of this is done live with no wait time or latency.
